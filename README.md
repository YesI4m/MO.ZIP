# MO.ZIP
## 동내 커뮤니티 웹 서비스

주위 이웃들과 네트워크를 만들 수 있는 동내 커뮤니티 웹서비스 '모집'



## 개발 팀
``
총 3명 (프론트엔드 2명 + 백엔드 1명)
``


## 기술 스택

- FrontEnd - React
- BackEnd - Spring Boot
- DB - MySQL
백엔드 API 작업 중 POSTMAN 사용

## 주요 기능

| 기능 | 설명 |
| ------ | ------ |
| 회원 | 회원가입, 로그인 |
| 게시글 | 게시글과 게시글의 댓글 작성, 수정, 삭제, 좋아요 가능 |
| hot 글 | 일주일간 사용자들의 많이 좋아요를 누른 글을 게시판 메인 페이지에서 확인 가능 |
| 파일 업로드 | 게시글에 파일첨부가능, 파일 첨부시 zip파일로 서버에 저장 |
| 사용자 페이지 | 사용자 페이지에서 사용자 정보 변경 가능 |
